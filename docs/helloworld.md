# Hello World

1. Change to the directory containing the source code for `helloworld`:

    ```bash
    cd src/helloworld
    ```

1. Compile `helloworld.c`:

    ```bash
    gcc -o helloworld helloworld.c
    ```

    An executable binary `helloworld` will be generated.

1. Execute the binary:

    ```bash
    ./helloworld
    ```
